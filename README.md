# Orientações Projeto

1. É necessário ter MySQL e o PHP (Xamp) instalados
2. O Projeto precisa estar no seu servidor PHP
3. Atualize as informações de conexão com o banco no config.php
4. Crie uma pasta chamada projeto-web ou com o nome que você quiser no htdocs do XAMP
5. Coloque o projeto nessa pasta
4. Rode a url do seu servidor do arquivo "createDb.php" no navegador para criar os bancos/tabelas
- Por padrão: http://localhost/projeto-web/src/back/createDb.php (ou no local de 'projeto-web', o nome da pasta que você criou no htdocs do XAMP)
5. Desfrute do melhor jogo de memória de pokemon da história
- Obs, seu link de inicio deve ficar parecido com algo assim: http://localhost/projeto-web/src/pages/index.php
